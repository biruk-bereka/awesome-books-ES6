# 📖 Awesome Books ES6 <a name="about-project"></a>

> This is an awesome books project which shows the books in the page, user can add new book and delete existing books. Buit using HTML, CSS and Javascript (ES6).

<!-- Features -->

### Key Features <a name="key-features"></a>
  - Create a collection where users can add a book with author and title.
  - User can remove the collection.
  - Collections are displayed at the top of the page.
  - Used local storage to store the collections.
  - User can navigate to different pages
